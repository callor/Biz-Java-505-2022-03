1.블록 레벨 텍스트 태그
: 혼자서 한 줄을 차지하여 너비가 100% 꽉 차게 표시되는 태그
1) h태그: heading태그 - 제목을 표시하는 태그로 h1~h6 이 있다

가장 큰 제목태그

가장 작은 제목태그


2) p태그: paragraph - 단락을 표시하는 태그로 줄바꿈없이 텍스트가 한 줄로 표시됨
텍스트의 길이가 브라우저 창의 너비보다 길면 자동으로 줄이 바뀐다.

br태그: 줄바꿈 처리는 태그로 닫는 태그가 없다

hr태그(horizontal) : 주제가 바뀌거나 할때 분기위전환을 위한 가로선 표시
닫는 태그가 없다.

div 태그(division): 영역을 나누는 블록 태그

2. 인라인 레벨 텍스트 태그

한 줄을 차지하지 않는 태그로 화면에 표시되는 컨텐츠만큼의 너비를 차지한다

굵은 텍스트

텍스트 일부만 스타일 적용하기
