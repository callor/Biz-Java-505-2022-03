여는태그와 닫는태그가 있고, 여는 태그만 사용하는 태그도 있다.
ip주소확인: ipconfig 로컬ip주소: 실제 ip주소, localhost, 127.0.0.1
html(HyperText Markup Language): 웹문서를 만드는 언어 웹문서의 기본골격(명사)
css(Cascading Style Sheet): 웹문서를 보기좋게 꾸미는 디자인표현(형용사/부사)
javascript/jQuery: 프로그래밍적 요소로 사용자의 행위에 의한 동작(동사)

1. 블록 레벨 텍스트 태그

태그 혼자서 한 줄을 차지하여 너비가 100% 로 꽉 차게 표현되는 태그

h태그 : h1>h2>h3>h4>h5>h6 - heading

p태그 : 단락을 표현하는 태그

hr태그 : 주제가 바뀌거나 분기위 전환을 할 때 사용하는 가로선

div태그: 영역을 나누는 블록태그

화면에 보이지는 않게 내부적으로 설명을 넣고자 할때 사용하는 것을 주석이라 한다.

2.인라인 레벨 텍스트 태그

컨텐츠만큼의 너비를 차지하는 태그

strong 태그: 굵게 표시하는 태그
span 태그: 텍스트의 일부만 특정스타일처리를 한다던가 할때 사용

  3. 메뉴3

시작값을 지정가능: start

4. 표를 만드는 태그

table 태그, tr(table row) 태그, td(table data) 태그
행열로 이루어진 표의 형태를 만들때 사용하는 태그
tr: 행에 해당하는 태그
td: 행 내부의 열에 해당하는 셀 태그
th(table head): 행 내부의 제목에 해당하는 셀 태그
